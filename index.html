About Me

Hi! I'm Ritik Kumar Saxena, a passionate Full Stack Web Developer with a strong interest in building clean, responsive, and user-friendly web applications. I specialize in the MERN stack (MongoDB, Express.js, React.js, Node.js) and have experience working with modern tools like Tailwind CSS and Framer Motion. I have 6 months of professional experience as an Application Engineer at Authentic Engineers, where I worked on real-time projects and strengthened my frontend and backend development skills. I've built projects ranging from an animated React portfolio and a real-time Weather App to a full-featured E-Commerce website. I'm always eager to learn, solve problems, and create impactful digital experiences that combine functionality with modern design.
